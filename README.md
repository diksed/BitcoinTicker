# Bitcoin Ticker

Bitcoin Ticker, canlı Bitcoin fiyatlarını görüntülemek için kullanılan bir Android uygulamasıdır.

## Özellikler

- Anlık Bitcoin fiyatlarını görüntüleme
- Detaylı Bitcoin bilgilerini görüntüleme
- Favori Bitcoinleri izleme
